Users

Every user has his own database-file. If you have just one user, trinitrotoluol will just start with this one without asking. If you have multiple users, trinitrotoluol will ask wich users database to be opened.

Hint:
The users are managed in a config-file located at:
  • Linux/Mac: ~/.trinitrotoluol/config.ini
  • Windows %appdata%\trinitrotoluol\config.ini
Everything else is stored in the according trinitrotoluol database-files.
